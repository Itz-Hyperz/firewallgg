FirewallGG Security

Welcome to FirewallGG! This is an API middleware that is allowing you to keep your Discord communities, Discord web based services, and much more secure to a new standard! FirewallGG is entirely open source and is accepting contributions right now! We are striving every day to redefine what a safe and secure community is on Discord. Interested in helping the project or contributing? Check out our GitHub for more information on the project! You can also see our Discord for community news, updates on FirewallGG, new bans that are now in place, and much much more!
